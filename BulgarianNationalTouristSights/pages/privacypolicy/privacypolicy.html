Privacy Policy

Using “National Places Of Bulgaria” is associated with an account. The purpose of this account is to save user notes and locations which users has marked as visited in order to persist them device independent. No additional data is saved by this application.

“National Places Of Bulgaria” App uses Internet connection in order to provide short information about each place.

“National Places Of Bulgaria” App does not record any personal information. When the user marks locations as visited users gps coordinates are transmitted for validation purposes.
